# 财务术语中英对照表

## 损益表
| 英文 | 繁中 |
|------|------|
| Revenue / Net Sales | 营收 / 净销售额 |
| Gross Profit | 毛利润 |
| Gross Margin | 毛利率 |
| Operating Income / EBIT | 营业利润 |
| Operating Margin | 营业利润率 |
| Net Income | 净利润 |
| EPS (Earnings Per Share) | 每股盈余 |
| Diluted EPS | 稀释每股盈余 |
| EBITDA | 息税折旧摊销前利润 |
| Non-GAAP / Adjusted | 非公认会计原则 / 调整后 |

## 现金流
| 英文 | 繁中 |
|------|------|
| Free Cash Flow (FCF) | 自由现金流 |
| Operating Cash Flow | 营运现金流 |
| CapEx (Capital Expenditure) | 资本支出 |
| Buyback / Share Repurchase | 股票回购 |

## 资产负债表
| 英文 | 繁中 |
|------|------|
| Inventory | 库存 |
| Deferred Revenue | 递延营收 |
| Days Inventory Outstanding (DIO) | 库存天数 |
| Cash & Equivalents | 现金及约当现金 |
| Net Debt | 净负债 |

| 英文 | 繁中 |

| Win Rate | 竞标胜率 |
| YoY (Year-over-Year) | 年增率 |
| QoQ (Quarter-over-Quarter) | 季增率 |

| Backlog | 积压订单 |
