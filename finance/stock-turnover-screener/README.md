# 📊 换手率翻倍选股 Skill

**stock-turnover-screener** — 筛选今日换手率较5日均值翻倍（≥2×）的美股个股，按板块分类，结合实时新闻解读催化剂，并给出评分与推荐建议。所有输出均为**简体中文**。

---

## 功能概览

- ✅ 从 Finviz 获取放量候选股（相对成交量 > 2×）
- ✅ 通过 Finnhub MCP Server 精确计算今日换手率 vs 5日均换手率
- ✅ 筛选换手率翻倍倍数 ≥ 2× 的个股
- ✅ 按 11 个 GICS 板块分类（信息技术、医疗保健、金融等）
- ✅ 搜索每只个股的最新新闻，标注催化剂类型（财报/并购/临床/宏观…）
- ✅ 综合评分（1–10分），输出重点关注 / 观察 / 中性 / 回避四级建议
- ✅ 全程简体中文输出，含免责声明

## 安装方法

### Claude.ai 网页版（推荐普通用户）

1. 下载本目录下的 [`stock-turnover-screener.skill`](./stock-turnover-screener.skill) 文件
2. 打开 [Claude.ai](https://claude.ai) → **Settings → Skills**
3. 点击 **Upload Skill**，选择下载的文件，安装完成 ✓

### Claude Code（精确数据模式）

需要配置本地 Finnhub MCP 服务器，可获得精确的换手率和5日历史量数据：

```bash
git clone https://github.com/yuzongg/claude-skills.git
cd claude-skills
cp .mcp.json.example .mcp.json   # 填入你的 Finnhub API Key 和路径
cd mcp-server && uv sync && cd ..
claude                           # 启动 Claude Code
```

详细配置步骤见 [mcp-server/README.md](../../mcp-server/README.md)

## 换手率定义

| 指标 | 计算方式 |
|------|---------|
| 今日换手率 | 今日成交量 ÷ 流通股数 × 100% |
| 5日均换手率 | 近5个完整交易日均量 ÷ 流通股数 × 100% |
| 翻倍倍数 | 今日换手率 ÷ 5日均换手率（≥2.0× 触发筛选） |

---

## 评分体系

| 评分因子 | 权重 | 说明 |
|----------|------|------|
| 换手率翻倍倍数 | 30% | surge_ratio 越高得分越高 |
| 新闻明确性 | 25% | 有明确正面催化剂得分高 |
| 价格走势 | 20% | 量价齐升为看涨确认 |
| 板块动能 | 15% | 同板块多只联动，加成 |
| 市值规模 | 10% | 市值越大可信度越高 |

| 等级 | 分数 | 含义 |
|------|------|------|
| 🟢 重点关注 | 7–10 | 催化剂明确，价格确认 |
| 🟡 观察 | 5–6.9 | 有吸引力但信号不明确 |
| ⚪ 中性 | 3–4.9 | 放量但无明显方向 |
| 🔴 回避 | <3 | 噪音、微盘炒作或负面催化剂 |

## 数据来源

| 数据 | 来源 | 说明 |
|------|------|------|
| 候选股票 + 今日成交量 | Finviz 筛选器 | 无需 API Key |
| 换手率精算（今日 vs 5日均） | Finnhub MCP + yfinance | 需配置 MCP 服务器 |
| 实时行情 | `mcp__finnhub__get_batch_quotes` | Finnhub /quote |
| 公司新闻 | `mcp__finnhub__get_news` | Finnhub /company-news |
