# 📊 换手率翻倍选股 Skill

**stock-turnover-screener** — 筛选今日换手率较20日均值翻倍（≥2×）的美股个股，按板块分类，结合实时新闻解读催化剂，并给出评分与推荐建议。所有输出均为**简体中文**。

---

## 功能概览

- ✅ 自动抓取当日异动数据（FinViz / StockAnalysis / TickFlow 等）
- ✅ 筛选换手率倍数 ≥ 2× 的个股
- ✅ 按 11 个 GICS 板块分类（信息技术、医疗保健、金融等）
- ✅ 搜索每只个股的最新新闻，标注催化剂类型（财报/并购/临床/宏观…）
- ✅ 综合评分（1–10分），输出重点关注 / 观察 / 中性 / 回避四级建议
- ✅ 全程简体中文输出，含免责声明

---

## 安装方法

### 下载安装（推荐）

1. 下载本目录下的 [`stock-turnover-screener.skill`](./stock-turnover-screener.skill) 文件
2. 打开 [Claude.ai](https://claude.ai) → **Settings → Skills**
3. 点击 **Upload Skill**，选择下载的文件
4. 安装完成 ✓

### 触发方式

安装后，在对话中说以下任意一种，Claude 都会自动调用此技能：

> "帮我筛一下今天换手率翻倍的股票"  
> "有哪些股票今天成交量异常放大？"  
> "做一个美股异动筛选报告"  
> `/stock-turnover-screener`

## 评分体系

| 评分因子 | 权重 |
|----------|------|
| 换手率涨幅 | 30% |
| 新闻明确性 | 25% |
| 价格走势 | 20% |
| 板块动能 | 15% |
| 市值规模 | 10% |

| 等级 | 分数 | 含义 |
|------|------|------|
| 🟢 重点关注 | 7–10 | 催化剂明确，价格确认 |
| 🟡 观察 | 5–6.9 | 有吸引力但信号不明确 |
| ⚪ 中性 | 3–4.9 | 放量但无明显方向 |
| 🔴 回避 | <3 | 噪音、微盘炒作或负面催化剂 |

## 注意事项

- 本技能依赖网络搜索获取实时数据，需在联网环境下使用
- 数据来源为公开筛选网站，可能存在延迟（通常 ≤15 分钟）
- 微盘股（市值 < 5000万美元）会被自动降权，防止炒作误导
- **本工具仅供参考，不构成投资建议**
