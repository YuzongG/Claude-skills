# 新闻→趋势→产业→个股 四阶漏斗

## 框架说明

本技能实现「早餐30分钟」操作法的核心逻辑——**从宏观新闻出发，逐层收窄，最终落到今日放量的具体标的**：

```
第一阶  事件   今天最重要的3条新闻是什么？
    ↓
第二阶  趋势   每条新闻背后是哪个趋势在推动？
    ↓
第三阶  产业   这些趋势会让哪些产业的订单/业绩/预期改变？
    ↓
第四阶  个股   这些产业中，哪几档今天放量且符合所有过滤条件？
```

---

## 语言要求

**所有输出使用简体中文。** 股票代码、公司英文名、数字保留原格式（如 NVDA、+12.4%）。新闻来源可保留英文。其余全部简体中文。

## 数据来源

| 数据 | 来源 | 说明 |
|------|------|------|
| 今日重要财经新闻 | `web_search` + `web_fetch` | 搜索当日头条财经新闻 |
| 候选股票列表、今日成交量 | Finviz（WebFetch） | 按产业关键词筛选放量股 |
| 换手率精算（今日/5日均值） | `mcp__finnhub__get_batch_turnover` | 精确过滤真正放量的股票 |
| 实时行情 | `mcp__finnhub__get_batch_quotes` | 价格、涨跌幅确认 |
| 公司资料 | `mcp__finnhub__get_profile` | 行业分类、市值验证 |
| 个股新闻 | `mcp__finnhub__get_news` | 确认个股是否有消息面催化剂 |

## 执行流程

### 第一阶 — 事件：抓取今日3条最重要新闻

#### Step 1.1 搜索今日财经头条

同时执行以下搜索，取最重要的3条：

```
web_search: "market moving news today [当前日期]"
web_search: "top financial news today US stocks"
web_search: "今日美股重要新闻 [当前日期]"
```

也可直接抓取财经聚合页：

```
web_fetch: https://finance.yahoo.com/
web_fetch: https://www.marketwatch.com/
```

#### Step 1.2 筛选标准

优先选择以下类型的新闻（按重要性排序）：

| 优先级 | 新闻类型 | 举例 |
|--------|---------|------|
| ⭐⭐⭐ | 宏观政策/央行/关税 | 美联储利率决议、新关税政策 |
| ⭐⭐⭐ | 地缘政治/大国博弈 | 半导体出口禁令、贸易谈判 |
| ⭐⭐ | 行业颠覆性事件 | AI重大突破、能源价格骤变 |
| ⭐⭐ | 标志性财报 | 巨头超预期/暴雷 |
| ⭐ | 监管政策变化 | FDA重大裁决、平台反垄断 |

**排除：** 纯公司内部人事变动、非市场相关社会新闻。

最终输出3条新闻，每条包含：**标题、来源、核心事实（一句话）**。

---

### 第二阶 — 趋势：分析每条新闻背后的驱动力

对每条新闻，识别其背后的**中长期趋势标签**：

| 趋势类别 | 代表关键词 |
|---------|-----------|
| 🤖 AI/算力 | 大模型、推理芯片、数据中心 |
| 🔋 能源转型 | 电动车、储能、太阳能、电网升级 |
| 🏭 制造回流 | 美国制造、供应链重组、在岸生产 |
| 💊 生物医疗 | GLP-1、基因疗法、AI制药 |
| 🛡️ 国防/安全 | 国防预算增加、网络安全 |
| 💱 利率/流动性 | 降息预期、信用利差、资金面 |
| 🌏 去全球化 | 关税、贸易壁垒、供应链多元化 |
| 🏠 房地产/消费 | 按揭利率、消费信心、零售数据 |

每条新闻分析输出：
- **新闻核心**：一句话总结
- **趋势标签**：1-2个标签
- **趋势逻辑**：为什么这条新闻是这个趋势的体现（2-3句话）
- **趋势持续性**：短期冲击（1-3天）/ 中期主题（1-3个月）/ 长期结构（1年+）

---

### 第三阶 — 产业：推导受影响的GICS板块与子行业

根据第二阶的趋势，列出**直接受益**和**间接受益**的产业，以及可能**受损**的产业：

#### 产业影响分析模板

```
趋势：[趋势名称]

直接受益产业：
- [产业名称]：[受益逻辑，一句话] → Finviz筛选关键词
- [产业名称]：[受益逻辑，一句话] → Finviz筛选关键词

间接受益产业：
- [产业名称]：[传导路径，一句话]

潜在受损产业：
- [产业名称]：[受损逻辑，一句话]
```

#### 常见趋势→产业映射

| 趋势 | 直接受益产业 | Finviz行业标签 |
|------|------------|--------------|
| AI/算力扩张 | 半导体、数据中心REIT、电力设备 | `ind_semiconductors`, `ind_reit` |
| 关税升级 | 美国本土制造商、原材料 | `ind_steelironore`, `ind_chemicals` |
| 降息预期 | 房地产、公用事业、高股息 | `ind_realestate`, `ind_utilities` |
| 国防预算增加 | 国防承包商、网络安全 | `ind_aerospacedefense` |
| GLP-1/减肥药 | 生物医药、医疗器械 | `ind_biotechnology`, `ind_medicaldevices` |
| 能源价格上涨 | 石油天然气、炼油商 | `ind_oilgas` |
| 贸易紧张 | 物流、港口、仓储 | `ind_airdeliveryfreight` |

**本步骤输出：** 最终确定 **2-4个重点产业**进入第四阶筛选。

---

### 第四阶 — 个股：筛选今日放量且符合过滤条件的股票

对第三阶确定的每个重点产业，执行以下筛选流程。

#### Step 4.1 — 从Finviz抓取候选股票

针对每个目标产业，构建Finviz筛选URL：

**基础放量筛选：**
```
web_fetch: https://finviz.com/screener.ashx?v=111&f=sh_relvol_o2,sh_avgvol_o200,cap_smallover&o=-relativevolume
```

**加入行业过滤（示例：半导体）：**
```
web_fetch: https://finviz.com/screener.ashx?v=111&f=ind_semiconductors,sh_relvol_o2,sh_avgvol_o200,cap_smallover&o=-relativevolume
```

常用行业标签参考：
- 半导体：`ind_semiconductors`
- 软件：`ind_softwareapplication`
- 生物医药：`ind_biotechnology`
- 石油天然气：`ind_oilandgas`
- 银行：`ind_banks`
- 国防：`ind_aerospacedefense`
- 电力：`ind_utilities`
- 零售：`ind_specialtyretail`

提取字段：**Ticker、公司名、价格、涨跌幅、今日成交量、市值**。

#### Step 4.2 — 精确换手率计算（MCP工具）

构建 `symbol_volumes` 字典，调用：

```
mcp__finnhub__get_batch_turnover(
  symbol_volumes={
    "NVDA": 52000000,
    "AMD": 38000000,
    ...
  }
)
```

**过滤条件（必须同时满足）：**

| 条件 | 阈值 | 说明 |
|------|------|------|
| 换手率倍数 | ≥ 2.0× | 今日换手率 ÷ 5日均换手率 |
| 市值 | ≥ $10亿 | 排除小盘炒作 |
| 股价 | ≥ $1 | 排除仙股 |
| 今日成交量 | ≥ 200K | 排除流动性极差标的 |

排序：按**换手率倍数**倒序，每个产业取 Top 5-8。

#### Step 4.3 — 验证与新闻确认

对筛选出的每只股票，调用新闻工具确认是否有消息面催化剂：

```
mcp__finnhub__get_news(symbol="NVDA", days_back=3)
```

**关键判断：** 个股放量是否能与第一阶新闻产生**逻辑链接**？

- ✅ **强链接**：个股新闻直接提及当日宏观事件（如「受关税政策影响，公司宣布扩大在美产能」）
- 🟡 **弱链接**：个股属于受益板块，但无直接相关新闻
- ❌ **无链接**：放量原因与当日主题无关（可能是独立催化剂，单独标注）

## 推荐等级标准

| 等级 | 得分区间 | 条件 |
|------|---------|------|
| 🟢 重点关注 | 7–10分 | 强逻辑链接 + 量价齐升 + 催化剂明确 |
| 🟡 观察 | 5–6.9分 | 弱链接或催化剂不清晰，但放量真实 |
| ⚪ 中性 | 3–4.9分 | 放量真实但与当日主题关联度低 |
| 🔴 回避 | < 3分 | 无逻辑链接、小盘炒作、或负面催化剂 |

---

## 评分体系

| 评分因子 | 权重 | 说明 |
|---------|------|------|
| 与当日新闻逻辑链接强度 | 35% | 强链接最高分，无链接最低分 |
| 换手率倍数 | 25% | 倍数越高得分越高 |
| 价格确认（涨跌幅） | 20% | 量价齐升为正向确认 |
| 市值与流动性 | 10% | 大市值可信度更高 |
| 板块共振 | 10% | 同产业多只股票同步放量加分 |

## 特殊情况处理

- **新闻面模糊（无明显主题）**：从宏观经济数据（PMI、非农、CPI）出发推导趋势
- **多条新闻指向同一产业**：合并分析，视为信号增强
- **第四阶无候选股**：说明产业当日无异常成交，建议观望或扩大换手率倍数至1.5×
- **个股独立催化剂**（与当日宏观无关）：单独在报告末尾列出「独立催化剂个股」板块
- **MCP工具返回error**：跳过该股票enrichment，标注「数据部分缺失」
- **ETF/SPAC**：默认排除
- **盘前/盘后时段**：注明，换手率倍数可能虚高，建议开盘后二次确认
