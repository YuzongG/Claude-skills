# WACC 计算详细说明

## 公式

```
ke = Rf + β × ERP
WACC = E/(E+D) × ke + D/(E+D) × kd × (1 − T)
```

## 参数说明

| 参数 | 含义 | 默认值 / 来源 |
|------|------|--------------|
| Rf | 无风险利率 | 美国10年期国债收益率（见 sector-defaults.md） |
| β | Beta 系数 | Finnhub profile → 若缺失用行业默认值 |
| ERP | 市场风险溢价 | 5.5%（Damodaran，2025） |
| ke | 股权成本 | CAPM 计算结果 |
| kd | 债务成本 | 5.0%（若无数据）/ 或用利息支出÷总债务 |
| T | 企业税率 | 21%（美国标准） |
| E | 股权市值 | market cap（百万美元） |
| D | 总债务 | 资产负债表长期债务 + 短期债务 |

## 三情景 WACC 调整逻辑

| 情景 | 调整 | 经济含义 |
|------|------|---------|
| 乐观 | WACC − 0.5% | 市场对公司风险重估偏低，融资成本下降 |
| 中性 | WACC 基准值 | 当前市场条件维持不变 |
| 悲观 | WACC + 1.0% | 宏观收紧、公司特定风险上升，融资成本提高 |

## 常见边界情况

- **β < 0.5**：取 β = 0.5（公用事业/必需消费品下限）
- **β > 3.0**：取 β = 3.0（极高风险上限）
- **净现金公司（D < 0）**：D 取 0，WACC = ke（全股权融资）
- **金融股**：D/E 结构特殊，WACC 不适用，改用 P/B 法
- **REIT**：因税务穿透结构，税率 T 取 0%

## 终值敏感性提示

终值（TV）通常占 EV 的 60–80%，对 WACC 和 g 极其敏感：

| WACC 变化 | 终值变化（约） |
|----------|--------------|
| +1% | −15% ~ −25% |
| −1% | +20% ~ +35% |

当终值占 EV 比例 > 80% 时，务必在报告中标注并建议用户做敏感性分析。
