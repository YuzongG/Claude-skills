# 拐点雷达（Pivot Radar）

> 定位：**上游 RS 选股（rs-leadership-scorer / rs-star-hunter）负责"谁强"，本技能负责"此刻扳机到没到、该不该动手"。**
> 别人管选股，我们管 **timing 执行**。

## 语言要求
**所有输出使用简体中文。** 股票代码、ETF 名（SPY/SMH/CIBR…）、数字百分比保留原格式。

---

## 灵魂原则（务必先向用户强调）

1. **看期望值，不看胜率。** 动量系统天生低胜率（30-40%），靠盈亏比取胜。别因胜率低而否掉正期望信号。
2. **强势/板块强弱都有有效期，必须滚动复评。** 信号是"此刻"快照；板块这周强不代表下周强（表2 已内建 1W 转弱自动跳过）。
3. **信号延迟一根确认。** 缠论分型要靠后一根 K 线确认，故信号针对的是"上一根"的拐点，无未来函数、不能当根实时追。

---

## 核心工作流

脚本在本技能 `scripts/` 目录，仅用 Python 标准库 + Yahoo Finance chart JSON（无需 pandas/yfinance）。

### 一键扫描（默认）
```
python3 <技能目录>/scripts/scan.py
```
自带 ~200 只 / 11 板块的 universe，直接输出两张表 + 板块 3M×1W 状态。**这是主入口。**

### 扫自选清单
```
python3 <技能目录>/scripts/scan.py NVDA,PANW,CRWD,...
```
只出表1（强势股入场），不做板块分类。

### 可选：对榜上标的深挖
- 催化剂/新闻：`mcp__finnhub__get_news` + WebSearch 交叉验证（严禁凭空编造，标源头+日期）。
- 个股 RS 深度定性：转 `rs-leadership-scorer` 技能。
- 若想换更新/更大的股票池：先跑 `scripts/sector_rs.py` 看板块强弱，再 WebFetch 强板块成分（`stockanalysis.com/etf/<ETF>/holdings/`）喂给 scan.py。

## 两张表与信号定义

### 📈 表1 · 强势股入场（Stage 2 龙头）
| 信号 | 定义 | 性格 | 回测（60天持有）|
|------|------|------|------|
| **双腿+RS≥B** | 抬高底(缠论higher-low) + 回踩重启(收回EMA5)，且个股 RS≥B | **胜率王** | 胜 61% / +6.1% |
| **RS翘头** | RS线(个股/SPY) 收回其10EMA 且 >50EMA（相对领导力恢复）| **利润王** | 胜 59% / +7.3% |

- **RS翘头是"相对强度择时"**，能在大盘跌、个股扛跌时就亮（价格没新高但 RS 已升），常领先价格。**别用价格过度过滤它**（回测证明加价格过滤反而变差）。
- 过热（Parabolic≥55）的标的**照常列出，标注"⚠️可能过热"**——是"别追高"的提示，不是删除。

### 📉 表2 · 板块驱动（弱转强 / 趋势延续）
先按 **3M 相对强度 × 1W 方向** 给板块分类，再对成分股用对应信号：

| 板块状态 | 用什么信号 | 抓什么 |
|---------|-----------|--------|
| 3M强 + 1W继续强（强势领先）| **SDUAL 点火**：站上EMA20 已缓慢转强 + 突然暴量(≥2.5×前3日均量) | 趋势中的暴量延续 |
| 3M弱 + 1W转强（弱转强）| **DUAL 筑底突破**：深调>25% + 新鲜从下方上穿EMA20 | 大圆底/缓跌后的突破 |
| 3M强 + 1W转弱（见顶转弱）| **跳过** | 别追正在见顶的强板块 |
| 3M弱 + 1W仍弱 | **跳过** | 别接飞刀 |

- **信号与板块是"分工"不是"一刀切"**：强板块（半导体型趋势）吃 SDUAL 点火；弱转强板块（生科/软件/成长型筑底）吃 DUAL 突破。DUAL 在半导体/能源无效，SDUAL 在防御板块无效——脚本已按板块状态自动选。

## 解读输出并给结论

把扫描结果翻译成投资语言，**务必包含**：
1. **两张表各自的候选**，标明每只的信号类型、RS 等级、板块、Parabolic（过热则提示别追）。
2. **优先级建议**：表1 双腿+RS≥B（高 RS 者）优先级最高；RS翘头按 RS 等级排序取前列；表2 结合板块状态。
3. **性质区分**：双腿+RS≥B 偏"高胜率快打"，RS翘头 偏"高利润"，SDUAL/DUAL 是板块驱动。
4. **风控提示**：结构止损参考底分型低点或 -2×ATR；分型延迟确认，实盘延后一根。
5. 一句操作提示 + 免责声明。

## 已知度量边界（大样本实测，务必诚实告知）

1. **回测为"纯持有 N 天不止损"，绝对收益含牛市 beta。** 可信的是**相对排名**（RS翘头 > 共振 > 价格双腿），而非绝对数字。加了止损后期望会收缩。
2. **RS≥B 过滤"配持有有用、配紧止损反而拖累"**：龙头回调会先摇摆（易被止损扫出），拿得住才恢复领先。
3. **信号强度 regime 依赖**：恢复段（如市场低点后）RS翘头/DUAL/SDUAL 明显更肥；趋势/做头段被稀释。
4. **表2 的板块→信号映射不写死**：每次按当前 3M×1W 实时判，板块轮动时自动切换。
5. **样本足够的结论**（可信）：双腿>单腿、RS翘头是利润王、双腁+RS≥B是胜率王、DUAL限筑底型板块、逃顶信号 regime 依赖（故本技能不做做空表）。
