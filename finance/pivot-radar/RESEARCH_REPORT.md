# 拐点雷达 · 研究报告

> 日期：2026-07-16
> 主题：从富途自建指标出发，设计并用大样本回测验证一套「强势股 timing 扫描器」（拐点雷达 skill）。
> 一句话：**别人管选股（谁强），本系统管 timing（此刻该不该进、用什么信号进）。**

---

## 一、方法论层面的四条硬结论

这四条贯穿所有信号，是整套系统的地基：

| 结论 | 说明 | 证据 |
|---|---|---|
| **看期望值，不看胜率** | 动量系统天生低胜率（30-40%），靠盈亏比取胜。期望 = 胜率×平均赢 − 败率×平均亏 | 全部信号胜率 31-40% 但期望为正；DUAL 曾因胜率被误判 |
| **所有阈值 ATR 归一化** | 涨幅/回档/乖离用 ATR 表达，否则高波动票被狂报、低波动票不报 | 图表指标搬进扫描器的核心改造 |
| **信号延迟一根确认** | 缠论分型要靠后一根 K 确认，无未来函数，不能当根实时追 | 富途版 REFX 未来函数已去除 |
| **信号强度 regime 依赖** | 恢复段（市场低点后）普遍翻倍，趋势/做头段被稀释 | 见下「自 2026-03」多组对比 |

## 二、各信号验证结果（大样本回测）

**测试池**：~200 只 / 11 板块，2 年日线；固定持有或统一止损出场；量基准 = 前 3 日均量。

### 1. 回踩重启（表1 主信号之一）
定义：底分型确认 + 抬高底(higher-low) + 回踩重启(收回 EMA5) + 趋势未破 + 回档 ≥ 1.2 ATR。两腿全中 = **双腿**，一腿 = **单腿**。

| 配置 | 出场方式 | 胜率 | 利润/期望 | 持仓 |
|---|---|---|---|---|
| 双腿 | 固定持有 60 天 | 54% | +5.0% | — |
| **双腿 + RS≥B** | 固定持有 60 天 | **61%** | **+6.1%** | — |
| 双腿 | 破 EMA20 出（带止损）| 33% | +0.34R | 10 天 |
| 双腿 + RS≥B | 破 EMA20 出 | 36% | +0.26R | 11 天 |

- **双腿 > 单腿**（稳健，跨板块成立）。
- **定位：胜率王**（双腿+RS≥B 固定持有 61% 胜）。

### 2. RS 回踩翘头（表1 主信号之二，来自朋友建议）
定义：RS 线（个股/SPY）收回其 10 EMA 且 > 50 EMA（相对领导力恢复）。

| 配置 | 胜率 | 利润 |
|---|---|---|
| RS翘头 全部（60 天持有）| **59%** | **+7.3%** |
| RS翘头（自 2026-03，60 天）| **69%** | **+22.3%** |
| RS翘头 + 价格>EMA50 过滤（60 天）| 57% | +6.3%（**更差**）|

- **定位：利润王**（利润全面高于回踩重启）。
- **关键洞察：RS 线领先价格**——加价格过滤反而变差，因为最肥的单子是「RS 已翘头、价格还没确认」的。**别用价格过度过滤 RS 信号。**

### 3. 共振（价格双腿 + RS 强）——已否决
| 信号 | 60 天胜率 | 利润 |
|---|---|---|
| RS翘头单独 | 59% | +7.3% |
| **共振** | 56% | +5.6% |
| 价格双腿单独 | 54% | +5.0% |

- 共振比价格双腿好，但**比 RS翘头单独差**。加价格约束把 RS翘头 的最佳单子过滤掉了。**结论：不叠，两个信号各出各的。**

### 4. DUAL 筑底突破（表2，弱转强）
定义：上穿 EMA20 + 量 ≥ 1.5×前3日均量 + 深调 >25% + 新鲜从下方（突破前 10 天 ≥6 天在 EMA20 下）。

| 板块（DUAL 深调+新鲜）| 期望 |
|---|---|
| 🟢 生科 | +1.06R |
| 🟢 软件 | +0.40R |
| 🟢 成长高波动 | +0.42R |
| 🟢 医疗 | +0.35R |
| 🔴 半导体 | +0.01R |
| 🔴 能源 | −0.45R |

- **DUAL 只在「筑底—爆发」型板块有效**（生科/软件/成长/医疗），在趋势/周期板块（半导体/能源）无效。
- **为何半导体无效**：过去两年半导体是趋势牛市、几乎不走大圆底，DUAL 无用武之地。

### 5. SDUAL 点火（表2，趋势延续）
定义：站上 EMA20 + E20 上行 + 已缓慢转强 + 今日量 ≥ 2.5×前3日均量（突然暴量首日）。

| 板块（SDUAL 点火）| 期望 |
|---|---|
| 🟢 半导体 | +0.59R |
| 🟢 生科 | +0.37R |
| 🟢 成长高波动 | +0.38R |
| 🔴 消费可选 | −0.39R |
| 🔴 金融/医疗 | ≈0 |

- **和 DUAL 互补**：半导体 DUAL 没用（+0.01）但点火有用（+0.59）——半导体不筑底突破、只暴量点火。

### 6. Parabolic 逃顶（表3，已砍掉）
定义：抛物线过热（曲率+加速+陡度+乖离，0-100）+ 顶分型/破 EMA5 扳机。

| 板块 | 触发后 10 日 | 是否有效 |
|---|---|---|
| 网络安全 | −3.2%（69% 下跌）| ✅ 有效 |
| 半导体 | +4.2%（仅 48% 下跌）| ❌ 失效 |

- **逃顶信号 regime 依赖**：牛市板块里做空被碾。**故砍掉逃顶/做空表，专注做多选股。**

## 三、关键发现与洞察

1. **RS 过滤的价值取决于出场方式**：
   - 配紧止损 → RS≥B **拖累**（+0.09R < RS<B 的 +0.26R），龙头回调先摇摆易被扫。
   - 配持有 → RS≥B **加分**（60 天 59% > 52%），龙头扛过回调后恢复领先。
2. **DUAL 没被淘汰，是站错了队**：它不属于龙头池（穿越信号在龙头上不触发），而属于「深调筑底型板块」——从「龙头榜」挪到「弱转强榜」后 +0.39R。
3. **筑底闸门过度工程化**：小样本（8 笔）显示「纠缠+窄幅+走平」让 DUAL 翻倍，但大样本（198 只）里几乎不触发（8% 命中）→ **不可用，已弃用**。是一次「小样本假象被大样本推翻」的教训。
4. **板块要看 3M×1W 二维**：3 个月强 + 这周转弱 = 见顶轮出，自动跳过（如半导体 3M+22 但 1W−2）。板块→信号映射**不写死**，每次按当前数据动态判，随板块轮动自动切换。

## 五、诚实的边界与警告

1. **回测多为「纯持有 N 天不止损」，绝对收益含牛市 beta**。可信的是**相对排名**（RS翘头 > 共振 > 价格双腿），不是绝对数字。加止损后期望会收缩。
2. **部分结论样本偏小**（如某些板块的分组），只能算「方向性」；筑底闸门就是被大样本推翻的例子。
3. **信号延迟一根、日线级、不做实时盘中**。
4. **强势/板块强弱都有有效期，任何扫描都是「此刻」快照，必须滚动复评。**
5. 本报告基于历史价量的客观规则，仅供研究参考，不构成投资建议。
